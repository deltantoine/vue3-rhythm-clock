# Vue3 Rhythm Clock

## Project setup
### Packages install
```sh
npm install
```

### Compiles and hot-reloads for development
```sh
npm run dev
```

### Compiles for production
```sh
npm run build
```

### Lint Script
```sh
npm run lint:script
```

### Lint Style
```sh
npm run lint:style
```

### Run your unit tests
```sh
npm run test
```
